# STOCK-PORTFOLIO-TRACKER
# 📈 Stock Portfolio Tracker

A simple Python-based stock tracking application that helps users monitor their investments by calculating the total value of owned shares across multiple stocks.

---

## 🚀 Features

* 📊 Tracks multiple stock holdings
* 💵 Calculates individual stock investment values
* 🧮 Computes total portfolio investment
* 📝 Saves portfolio summary to a text file
* ⚠️ Handles invalid user input gracefully
* 🖥️ Beginner-friendly command-line interface

---

## 🛠️ Technologies Used

* Python
* Dictionaries
* File Handling
* Exception Handling
* User Input Processing

---

## 📂 Supported Stocks

* AAPL (Apple)
* GOOGL (Google)
* TSLA (Tesla)
* AMZN (Amazon)
* INFY (Infosys)

---

## 📈 Working Principle

1. Displays available stocks and prices.
2. User inputs owned share quantities.
3. Program calculates:

   * Individual investment per stock
   * Total portfolio value
4. Displays results in terminal.
5. Saves complete summary to `portfolio_summary.txt`.
